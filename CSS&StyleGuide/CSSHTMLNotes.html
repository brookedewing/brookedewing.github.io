CSS Color Formatting

Color is a very important factor in making beautiful websites. Below is a quick overview of some of the options you can use for coding colors on your own webpage! Also, here is a great website to reference when trying to decide what colors you want to use! (Warning: when using this website, I had trouble with HSL values showing incorrectly and wound up using a separate converter, so keep that in mind if using this format).

Name CSS Input Color Output Notes
Named Colors background-color: purple; The benefit of this format is that it is a very simple way to start using colors! There are around 140 options to choose from and most text editors will autofill as you begin to type a color. However, the simplicity can also be a drawback as it limits the creative choices you have.
Hex (Hexadecimal) Codes background-color: #8DDED1; This format utlizes a base 16 system instead of base 10 and uses letters A-F to represent the additional values. The format is a hash sign followed by six digits (each pair representing red, green and blue in that order). 00 is the darkest version of the color (closest to black) and FF is the lightest version (closest to white). If the pairs are repeating values, it can be represented as a three digit number instead (e.g. #00FF00 becomes #0F0). I've created a sheet in Excel to better show the conversion from decimal to hex and have linked that here
RGB (red, green, blue) background-color: rgb(65, 130, 15); Each color is represented by a decimal between 0-255, with 0 being the closest to black and 255 the closest to white NEED TO ADD NOTES
HSL (Hue, Saturation, Lightness) background-color: hsl(17, 91%, 65%); HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is). Hue ranges from 0 to 360 and saturation and lightness are both represented as percentages (NOTE: this explanation was taken directly from Codecademy - Nov 2022)

Alpha Value

Another way to format your color is by changing the transparency - this can be done using the alpha value. See the table below for examples on how to include this value. Each of these are shown at 75% opacity as compared to the colors above.

Color Format How to Add Alpha CSS Input Color Output
Named Colors This format just utilizes another tag "Opacity" where you can set the value between 0 and 1 (0 being transparent and 1 being opaque) background-color: purple;
opacity: 0.75;
Hex Two digits added to end of string - NEED TO ADD NOTES background-color: #8DDED1BF;
RGB Using rgba instead of rgb, add the alpha value at the end of the string either in decimal form (0 being fully transparent and 1.0 being fully opaque) or as a percentage (0%-100%). background-color: rgba(65, 130, 15, 0.75);
HSL NEED TO ADD NOTES background-color: hsla(17, 91%, 65%, 75%);
